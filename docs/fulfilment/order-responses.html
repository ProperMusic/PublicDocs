Order Response Message

Example Data​


 <?xml version="1.0" encoding="UTF-8" ?>
 <orderResponse>
 <customerId>#####</customerId>
 <orderId>#####</orderId>
 <storeId>#####</storeId>
 <storeOrderId>$$$$$</storeOrderId>
 <storeCustomerId>$$$$$</storeCustomerId>
 <statusString>order held. reason : incomplete</statusString>
 <orderLines>
     <orderLine>
         <catalogueNumber>$$$$$</catalogueNumber>
         <qty>1</qty>
         <price>10.99</price>
         <properLineId>#####</properLineId>
         <lineId>$$$$$</lineId>
         <lineStatuses>
            <status>
                <statusCode>0x00000190</statusCode>
                <statusMessage>Line Rejected. See errors for reason</statusMessage>
            </status>
         </lineStatuses>
         <lineErrors>
            <error>
                <errorCode>0x0000138D</errorCode>
                <errorMessage>Could not be found using catalogue number: No Match</errorMessage>
            </error>
         </lineErrors>
     </orderLine>
 </orderLines>
 </orderResponse>

resp.[original file name].xml.log

ValueFormatDescription
customerIdintegerProper's Internal Customer Id

storeOrderIdtextThe Client's Order Id
storeCustomerIdtextThe Client's Customer Id
statusStringtextSee Order level Statuses
